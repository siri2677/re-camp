# Re:Camp Art References

이 디렉터리는 Re:Camp 아트 제작의 기준 이미지와 승인된 레퍼런스를 관리한다.

## 현재 포함된 파일

### `art_direction/ReCamp_Character_Lineup.png`

루나, 미유, 코코, 이리스, 노아의 역할·색상·실루엣을 통합한 1차 캐릭터 라인업 콘셉트다.

- 상태: REVIEW
- 용도: 캐릭터 방향 통일, 개별 제작 시트의 출발점
- 제한: 최종 원화, Turnaround, 3D 모델링 시트로 직접 사용하지 않음

## 사용 우선순위

```text
개별 캐릭터 APPROVED 제작 시트
> APPROVED 아트 디렉션 보드
> REVIEW 통합 라인업
> WIP 생성 후보
```

## 운영 원칙

- 대량 생성된 모든 후보 이미지를 저장하지 않는다.
- 캐릭터별 얼굴·헤어·의상·무기 구조는 Approved 개별 시트에서 확정한다.
- 파일을 교체할 때 버전과 상태를 갱신한다.
- AI 생성 이미지는 도구, 생성일, 입력 레퍼런스, 사람의 수정 내역을 기록한다.
- 상업 사용 조건이 확인되지 않은 생성물은 최종 게임 자산으로 사용하지 않는다.
