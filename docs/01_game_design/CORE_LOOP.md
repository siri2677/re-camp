# CORE_LOOP.md

## 핵심 루프 개요

Re:Camp의 핵심 루프는 **탐험 → 전투 → 자원 획득 → 귀환 → 캠프 성장 → 재탐험**이다.

```text
탐험 준비
→ 위험 구역 진입
→ 몬스터 전투
→ 자원 획득
→ 랜덤 스킬 선택
→ 귀환 판단
→ 결과 정산
→ 캠프 성장
→ 다음 탐험
```

## 단계별 플레이어 행동

| 단계 | 플레이어 행동 | 보상/결과 |
|---|---|---|
| 탐험 준비 | 캐릭터/장비/스킬 확인 | 전략 선택 |
| 위험 구역 진입 | 스테이지 시작 | 탐험 시작 |
| 전투 | 이동, 회피, 공격 | 몬스터 처치 |
| 자원 획득 | 드랍 자원 수집 | 성장 재료 확보 |
| 스킬 선택 | 랜덤 스킬 중 선택 | 전투 방식 변화 |
| 귀환 판단 | 더 탐험할지 귀환할지 결정 | 리스크/보상 선택 |
| 결과 정산 | 획득 자원 확인 | 보상 확정 |
| 캠프 성장 | 시설 강화 | 다음 탐험 강화 |

## 리스크/보상 구조

탐험 시간이 길어질수록 보상은 커지지만 위험도도 상승한다.

```text
초반: 안전하지만 보상 적음
중반: 보상과 위험 균형
후반: 보상 많지만 사망 위험 큼
```

## 귀환 시스템

귀환은 Re:Camp의 핵심 선택 요소다.

- 귀환 성공: 획득한 자원 보존
- 사망: 일부 자원 손실
- 제한 시간 종료: 강제 귀환 또는 보스 출현

## 반복 플레이 목표

플레이어는 매 판 아래 목표 중 하나 이상을 달성하게 한다.

```text
자원 수집
스킬 조합 실험
보스 처치
캠프 시설 강화
캐릭터 성장
더 높은 위험도 도전
```

## MVP 기준 핵심 루프

MVP에서는 아래 루프만 먼저 구현한다.

```text
로비
→ 전투 시작
→ 몬스터 처치
→ 자원 획득
→ 결과 화면
→ 캠프 시설 강화
→ 로비 복귀
```
