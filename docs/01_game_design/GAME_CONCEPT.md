# Re:Camp Game Concept

## 1. 게임명

Re:Camp

## 2. 장르

- 3D 고정 쿼터뷰 로그라이트 RPG
- 생존 탐험과 귀환
- 캠프 거점 성장
- 서브컬쳐 캐릭터 수집·성장
- Steam Windows 프리미엄 싱글플레이

## 3. 한 줄 소개

멸망 이후 자연에 잠식된 위험 구역을 3~5분 동안 탐험하고 자원을 회수해 생존자 캠프와 캐릭터를 성장시키는 로그라이트 RPG.

## 4. 기획 의도

```text
짧은 탐험의 긴장감
+ 더 탐험할지 귀환할지 선택하는 리스크·보상
+ 캠프가 성장하는 누적 성취감
+ 매력적인 캐릭터를 직접 조작하는 애착
```

플레이어는 위험 구역에서 몬스터를 피하거나 처치하고 자원을 수집한다. 오래 머물수록 보상 가능성과 위험도가 함께 증가한다. 무사히 귀환하면 획득 자원을 캠프 시설에 투자해 다음 탐험을 더 유리하게 준비한다.

## 5. 핵심 재미

1. 제한된 시간 안에 경로와 전투를 판단하는 탐험
2. 자원을 더 모을지 안전하게 귀환할지 결정하는 선택
3. 공격·회피·스킬 조합으로 달라지는 로그라이트 전투
4. 캠프 시설이 실제 플레이 편의와 전투력으로 이어지는 성장
5. 외형·성격·역할이 분명한 서브컬쳐 캐릭터에 대한 애착
6. 여러 번의 짧은 탐험이 하나의 장기 성장 세션으로 연결되는 반복성

## 6. 플레이 시간 설계

한 판은 짧게 유지하지만 PC 게임 전체 경험은 짧지 않게 구성한다.

```text
1회 탐험: 3~5분
캠프 정비: 1~3분
권장 플레이 세션: 탐험 3~6회, 약 20~40분
```

이를 위해 본편에는 다음 장기 동기를 제공한다.

- 캐릭터 해금
- 캠프 시설 성장
- 캐릭터별 빌드
- 난이도 단계
- 도전 목표·업적
- 친밀도·대화·관계 진행

## 7. 플레이어 경험 목표

| 경험 | 설명 |
|---|---|
| 긴장감 | 탐험 시간이 지날수록 적과 위험이 증가한다. |
| 선택 | 귀환, 추가 탐험, 위험한 보상 중 하나를 선택한다. |
| 숙련 | 이동·회피·스킬 선택과 적 패턴 이해가 향상된다. |
| 성장 | 캐릭터와 캠프가 반복 플레이를 통해 강화된다. |
| 애착 | 캐릭터의 외형·모션·대사·역할을 기억하게 된다. |
| 반복성 | 짧은 탐험 결과가 장기 목표와 다음 세션으로 이어진다. |

## 8. 플랫폼과 판매 경험

### Steam Windows

- 1차 개발·상용 출시 플랫폼
- Keyboard/Mouse와 Gamepad 직접 조작
- 1920×1080 기준 UI와 전투 가독성
- 프리미엄 1회 구매
- 무료 Demo 제공
- 가챠·광고·유료 재화 없음

Steam을 우선하는 이유는 Re:Camp가 서버와 라이브 운영 없이 완결 가능한 패키지 게임 구조이기 때문이다.

### itch.io

- Alpha·외부 테스트·제한 공개 빌드
- Steam Demo 이전 설치·입력·성능 검증

### Epic Games Store

- Steam 출시 안정화 후 2차 PC Store로 검토

### Android

- 짧은 탐험 구조는 모바일에 적합하지만 현재 프리미엄 패키지 모델과 3D 품질을 우선한다.
- Steam Full Release 이후 Touch UI·성능·유료 판매 모델을 별도로 검증한다.
- 모바일 Port를 위해 가챠·광고·F2P 라이브 운영으로 전환하지 않는다.

## 9. 비주얼 톤

```text
2D 일반 등신 캐릭터 일러스트
+ 3D 5~6등신 스타일라이즈드 인게임 모델
+ 툰 셰이딩
+ 자연에 잠식된 파스텔 폐허
+ 따뜻한 캠프와 위협적인 탐험 지역의 대비
```

캐릭터의 매력과 식별성을 배경의 사실성보다 우선한다.

## 10. 캐릭터 방향

플레이어블 캐릭터는 루나·미유·코코·이리스·노아 5명이다. 현재 통합 라인업은 역할·색상·실루엣의 Direction Approved 상태이며 개별 최종 제작 시트는 아직 제작 전이다.

루나 Character Proof를 먼저 진행한 뒤 검증된 Base Mesh·Rig·Shader·Prefab을 나머지 캐릭터에 확장한다.

## 11. 차별화 포인트

```text
직접 조작하는 짧은 로그라이트 전투
+ 탐험 후 귀환하는 명확한 리스크·보상
+ 캠프 성장으로 이어지는 장기 목표
+ 2D 서브컬쳐 디자인을 3D 쿼터뷰 플레이로 연결
+ 폐허와 희망적인 캠프의 정서적 대비
+ 모바일식 짧은 한 판을 PC 패키지의 장기 성장으로 재구성
```

## 12. 1차 타깃

- 짧은 런과 장기 성장 모두를 원하는 Steam 로그라이트 유저
- 서브컬쳐 캐릭터와 직접 조작 전투를 선호하는 유저
- 생존·탐험·귀환 구조를 좋아하는 유저
- 복잡한 MMORPG보다 반복 가능한 소규모 RPG를 원하는 유저
- Gamepad나 휴대형 PC에서도 짧게 플레이하고 싶은 유저

## 13. 판매 기준

```text
플랫폼: Steam Windows
판매 방식: 프리미엄 1회 구매
기준 정가: 11,000원
검토 범위: 9,900~13,500원
출시 할인 가정: 10%
무료 Demo: 제공
```

가격은 실제 콘텐츠 분량과 외부 테스트 결과를 확인한 후 확정한다.

- 오픈월드
