# Re:Camp Game Concept

## 1. 게임명

Re:Camp

## 2. 장르

- 3D 고정 쿼터뷰 로그라이트 RPG
- 생존 탐험과 귀환
- 캠프 거점 성장
- 서브컬쳐 캐릭터 수집·성장

## 3. 한 줄 소개

멸망 이후 자연에 잠식된 위험 구역을 3~5분 동안 탐험하고 자원을 회수해 생존자 캠프를 성장시키는 로그라이트 RPG.

## 4. 기획 의도

Re:Camp는 대규모 오픈월드나 콘텐츠 양보다 다음 경험에 집중한다.

```text
짧은 세션의 긴장감
+ 더 탐험할지 귀환할지 선택하는 리스크·보상
+ 캠프가 성장하는 누적 성취감
+ 매력적인 캐릭터를 직접 조작하는 애착
```

플레이어는 위험 구역에서 몬스터를 피하거나 처치하고 자원을 수집한다. 오래 머물수록 보상 가능성과 위험도가 함께 증가한다. 무사히 귀환하면 획득 자원을 캠프 시설에 투자해 다음 탐험을 더 유리하게 준비한다.

## 5. 핵심 재미

1. 제한된 시간 안에 경로와 전투를 판단하는 탐험
2. 자원을 더 모을지 안전하게 귀환할지 결정하는 선택
3. 공격·회피·스킬 조합으로 달라지는 로그라이트 전투
4. 캠프 시설이 실제 플레이 편의와 전투력으로 이어지는 성장
5. 외형·성격·역할이 분명한 서브컬쳐 캐릭터에 대한 애착

## 6. 플레이어 경험 목표

| 경험 | 설명 |
|---|---|
| 긴장감 | 탐험 시간이 지날수록 적과 위험이 증가한다. |
| 선택 | 귀환, 추가 탐험, 위험한 보상 중 하나를 선택한다. |
| 숙련 | 이동·회피·스킬 선택과 적 패턴 이해가 향상된다. |
| 성장 | 캐릭터와 캠프가 반복 플레이를 통해 강화된다. |
| 애착 | 캐릭터의 외형·모션·대사·역할을 기억하게 된다. |
| 반복성 | 짧은 세션마다 다른 스킬과 자원 결과를 경험한다. |

## 7. 플랫폼 경험

### Windows PC

- 1차 개발 및 포트폴리오 기준
- Keyboard/Mouse와 Gamepad 직접 조작
- 1920×1080 기준 UI와 전투 가독성 우선

### Android

- PC Vertical Slice 이후 확장
- 가상 조이스틱과 터치 버튼
- 동일한 게임 규칙과 데이터 사용
- 별도의 성능·UI 품질 단계 제공

## 8. 비주얼 톤

```text
2D 일반 등신 캐릭터 일러스트
+ 3D 5~6등신 스타일라이즈드 인게임 모델
+ 툰 셰이딩
+ 자연에 잠식된 파스텔 폐허
+ 따뜻한 캠프와 위협적인 탐험 지역의 대비
```

캐릭터의 매력과 식별성을 배경의 사실성보다 우선한다.

## 9. 캐릭터 방향

플레이어블 캐릭터는 루나·미유·코코·이리스·노아 5명이다. 현재 통합 라인업은 역할·색상·실루엣의 방향 승인 상태이며 개별 최종 제작 시트는 아직 제작 전이다.

최종 제작은 루나 Character Proof를 먼저 진행한 뒤 검증된 Base Mesh·Rig·Shader·Prefab을 나머지 캐릭터에 확장한다.

## 10. 차별화 포인트

```text
직접 조작하는 짧은 로그라이트 전투
+ 탐험 후 귀환하는 명확한 리스크·보상
+ 캠프 성장으로 이어지는 장기 목표
+ 2D 서브컬쳐 디자인을 3D 쿼터뷰 플레이로 연결
+ 폐허와 희망적인 캠프의 정서적 대비
```

## 11. 1차 타깃

- 짧은 시간에도 성장과 완결감을 원하는 RPG 유저
- 서브컬쳐 캐릭터와 직접 조작 전투를 선호하는 유저
- 로그라이트·생존·탐험·귀환 구조를 좋아하는 유저
- 복잡한 MMORPG보다 반복 가능한 소규모 RPG를 원하는 유저

## 12. 초기 제외 요소

- 멀티플레이, PvP, 길드
- 실시간 계정 서버
- 가챠와 결제 상점
- 오픈월드
- 복잡한 장비 파밍과 의상 교체
- 대량 컷신과 풀 보이스
