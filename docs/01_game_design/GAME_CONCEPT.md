# GAME_CONCEPT.md

## 게임명

Re:Camp

## 장르

- 모바일 로그라이트 RPG
- 생존 탐험
- 거점 성장
- 서브컬쳐 캐릭터 수집/성장

## 한 줄 소개

멸망한 세계에서 생존자 캠프를 성장시키기 위해 5분 동안 위험 구역을 탐험하고 자원을 회수하는 로그라이트 생존 RPG.

## 기획 의도

Re:Camp는 대규모 오픈월드나 고사양 그래픽보다 **짧은 세션, 반복 성장, 캐릭터 애착, 귀환 성취감**에 집중한다.

플레이어는 위험 구역에 들어가 자원을 수집하고, 몬스터를 피하거나 처치하며, 적절한 타이밍에 귀환해야 한다. 귀환에 성공하면 자원을 사용해 캠프를 성장시키고 다음 탐험을 더 유리하게 준비할 수 있다.

## 핵심 재미

```text
1. 짧은 시간 안에 위험 구역을 탐험하는 긴장감
2. 자원을 들고 무사히 귀환했을 때의 성취감
3. 캠프가 점점 성장하는 누적 성장감
4. 귀여운 서브컬쳐 캐릭터를 수집하고 성장시키는 만족감
5. 매 판 다른 스킬 선택과 전투 흐름
```

## 플레이어 경험 목표

| 경험 | 설명 |
|---|---|
| 긴장감 | 탐험 시간이 지날수록 위험도가 증가한다. |
| 선택 | 귀환할지 더 탐험할지 선택한다. |
| 성장 | 캐릭터와 캠프가 누적 성장한다. |
| 애착 | 캐릭터 외형, 성격, 역할이 명확하다. |
| 반복성 | 짧은 세션을 여러 번 반복해도 보상이 있다. |

## 비주얼 톤

```text
귀여운 서브컬쳐 캐릭터
+ 부드러운 포스트 아포칼립스
+ 파스텔 톤 폐허
+ 모바일 친화적 UI
```

## 타깃 유저

- 짧은 시간에 성장감을 느끼고 싶은 모바일 RPG 유저
- 귀여운 캐릭터와 수집 요소를 좋아하는 유저
- 로그라이트/생존/탐험 구조를 선호하는 유저
- 복잡한 MMORPG보다 가볍고 반복 가능한 RPG를 원하는 유저

## 차별화 포인트

```text
직접 조작 로그라이트 전투
탐험 후 귀환하는 리스크/보상 구조
캠프 성장 기반의 장기 목표
서브컬쳐 캐릭터와 생존 세계관의 조합
```
