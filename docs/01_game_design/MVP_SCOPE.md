# MVP_SCOPE.md

## MVP 목표

Re:Camp의 MVP는 전체 게임 완성이 아니라 **핵심 플레이 루프가 실제로 작동하는지 검증하는 것**이다.

## MVP 핵심 루프

```text
로비
→ 탐험 준비
→ 전투 진입
→ 몬스터 처치
→ 자원 획득
→ 귀환/사망
→ 결과 정산
→ 캠프 성장
→ 로비 복귀
```

## 포함 범위

| 구분 | MVP 범위 |
|---|---|
| 캐릭터 | 1명 |
| 조작 | 모바일 가상 조이스틱 / 키보드 테스트 입력 |
| 공격 | 기본 자동 공격 또는 반자동 공격 |
| 스킬 | 6개 |
| 몬스터 | 일반 3종 |
| 보스 | 1종 |
| 스테이지 | 폐허 거리 1개 |
| 자원 | 고철, 식량, 데이터 조각 |
| 캠프 시설 | 발전소, 작업대, 식량 창고 |
| UI | 로비, 전투, 결과, 캠프 |
| 저장 | 로컬 저장 |

## 완료 기준

MVP는 아래 조건을 만족하면 완료로 본다.

```text
1. 플레이어가 전투 스테이지에 진입할 수 있다.
2. 플레이어가 이동할 수 있다.
3. 몬스터가 스폰되고 플레이어를 추적한다.
4. 플레이어가 몬스터를 공격할 수 있다.
5. 몬스터가 사망하면 자원을 드랍한다.
6. 플레이어가 자원을 획득할 수 있다.
7. 전투가 종료되면 결과 화면이 나온다.
8. 획득한 자원으로 캠프 시설을 강화할 수 있다.
9. 강화 결과가 저장된다.
10. 다시 전투에 진입할 수 있다.
```

## MVP 우선순위

```text
P0: 씬 흐름, 플레이어 이동, 몬스터 스폰, 기본 전투
P1: 자원 획득, 결과 화면, 캠프 성장
P2: 스킬 선택, 보스전, 저장/불러오기
P3: 연출, 사운드, 튜토리얼, UI 개선
```
