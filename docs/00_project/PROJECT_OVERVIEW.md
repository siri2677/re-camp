## 프로젝트명

Re:Camp

## 한 줄 소개

Re:Camp는 멸망한 세계에서 생존자 캠프를 성장시키기 위해 위험 구역을 탐험하고 자원을 회수하는 5분 생존 로그라이트 RPG이다.

## 프로젝트 목적

- Google Play Android를 1차 제품 목표로 하는 Unity 모바일 로그라이트 RPG 개발
- Windows Unity Editor 기반 개발·자동 테스트와 Android APK·실기기 검증 분리
- 직접 조작 기반의 탐험/전투/귀환 루프 검증
- 서브컬쳐풍 캐릭터와 부드러운 포스트 아포칼립스 감성 확립
- 포트폴리오 및 확장 가능한 게임 IP 기반 마련

## 핵심 개발 목표

```text
1. 플레이어 이동
2. 몬스터 스폰
3. 기본 공격
4. 자원 획득
5. 결과 화면
6. 캠프 성장
7. 저장/불러오기
```

## 1차 MVP 범위

| 항목 | 범위 |
|---|---:|
| 캐릭터 | 1명 |
| 스테이지 | 1개 |
| 몬스터 | 3종 |
| 보스 | 1종 |
| 스킬 | 6개 |
| 아이템/자원 | 10개 내외 |
| 캠프 시설 | 3개 |
| 1판 플레이 시간 | 3~5분 |

## 제외 항목

MVP에서는 아래 항목을 제외한다.

```text
실시간 멀티플레이
길드
PvP
가챠
대규모 오픈월드
복잡한 컷신
음성 더빙
시즌 이벤트
```

## 개발 방향

초기 목표는 큰 콘텐츠 양이 아니라 **전투 1판이 완전히 돌아가는 구조**를 만드는 것이다.

```text
로비 → 탐험 준비 → 전투 → 결과 → 캠프 성장 → 로비
```
