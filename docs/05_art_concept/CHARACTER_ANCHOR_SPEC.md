# Re:Camp Character Anchor Specification

이 문서는 `ART-0102`의 도구 비종속 캐릭터 일관성 기준이다. 이미지 생성 도구, Figma,
Blender, Unity가 달라도 루나·미유·코코·이리스·노아를 같은 캐릭터로 유지하기 위해 사용한다.
캐릭터 설정은 `CHARACTER_BIBLE.md`, 공통 표현은 `ART_DIRECTION.md`가 우선한다.

`ReCamp_CharacterLineup_REVIEW_v003.png`는 아래 규칙을 점검하는 후보일 뿐이며 승인 원화가
아니다. 사람 승인을 받은 개별 제작 시트가 생기면 해당 시트가 얼굴·의상·장비 구조의 최우선
앵커가 된다.

## 1. 앵커 강도

| 등급 | 의미 | 변경 조건 |
|---|---|---|
| `LOCKED` | 캐릭터 정체성 또는 제품 방향 | 사용자 결정이나 Character Bible 개정 필요 |
| `CONTROLLED` | 같은 캐릭터로 읽히는 범위 안에서 조정 가능 | Review 이력과 변경 이유 기록 |
| `OPEN` | 포즈·표정·세부 배치처럼 제작물별 변형 가능 | 상위 앵커와 충돌하지 않으면 자유 |
| `FORBIDDEN` | 역할 혼동·성별·비율·세계관 회귀를 일으킴 | 제작 후보에서 제외 |

## 2. 전 캐릭터 공통 `LOCKED`

- 플레이어블은 정확히 루나·미유·코코·이리스·노아 5명의 성인 여성이다.
- 2D 대표 이미지는 일반 등신, 인게임 3D는 5~6등신 스타일라이즈드다. 기본 목표는 약 5.2~5.5등신이다.
- 3D는 공용 Humanoid 골격과 공용 이동·피격 애니메이션을 사용할 수 있어야 한다.
- 머리·상체·대표 장비 실루엣만으로 Orthographic 쿼터뷰에서 구분돼야 한다.
- 각 캐릭터는 대표 특징 1개와 보조 특징 2개를 우선하고 미세 장식을 늘리지 않는다.
- 캐릭터별 대표색·보조색·포인트색과 역할 장비를 서로 바꾸지 않는다.
- 얼굴 전체를 가리는 헬멧, 2~3등신 유아형 치비, 현실적인 6~7등신, 과도한 실사·밀리터리·고어를 사용하지 않는다.
- 생성 결과는 `WIP` 또는 `REVIEW`이며 사람 승인 전 `APPROVED`로 승격하지 않는다.

## 3. 공통 제작 변수

모든 후보와 제작 시트의 동반 메타데이터에 다음 필드를 기록한다.

```yaml
Anchor Spec Version: 1.0
Character ID:
Base Review Asset:
Approved Sheet: None
Face Anchor:
Hair Anchor:
Body / Silhouette Anchor:
Outfit Anchor:
Equipment Anchor:
Palette Anchor:
Material Anchor:
Allowed Variations:
Rejected Drift:
Human Edits:
Reviewer: Pending
```

얼굴·헤어·의상 후보를 교체할 때는 이전 후보와 나란히 비교하고 `Face Anchor`, `Hair Anchor`,
`Outfit Anchor` 변경 여부를 명시한다. Seed나 모델명처럼 확인할 수 없는 값은 추정하지 않는다.

## 4. CH001 루나

| 범주 | 앵커 |
|---|---|
| 역할 인상 | 빠르고 장난스럽지만 임무 중 판단이 빠른 정찰형 근접 딜러 |
| 얼굴 | 성인 여성으로 읽히는 밝고 영리한 인상, 장난기와 집중 상태의 대비 |
| 헤어 | 민트 계열의 짧거나 중간 길이, 후드 밖에서 빠른 움직임이 읽히는 가벼운 덩어리 |
| 대표 실루엣 | 고양이 귀 후드 `LOCKED`; 짧은 재킷과 활동적인 하체; 다섯 명 중 가장 가벼운 질량감 |
| 장비 | 에너지 단검 `LOCKED`; 탐지 장치와 소형 백팩; 장비 발광은 민트·청록 중심 |
| 팔레트 | 민트 / 크림 / 핑크 |
| 대표 특징 | 고양이 귀 후드 |
| 보조 특징 | 에너지 단검, 탐지 장치 |
| `CONTROLLED` | 노출도, 재킷 길이, 헤어 세부, 단검 날 형태, 백팩 크기 |
| `FORBIDDEN` | 의료 십자·간호 모자·원형 의료 방패처럼 코코로 읽히는 모티프; 장총·대형 방패 |

`REVIEW v003`의 루나에 반복된 의료 십자 표시는 코코의 역할 언어와 충돌하므로 후속 단독
후보에서 제거한다. 고양이 후드, 민트 헤어, 단검, 탐지 장치의 네 요소를 우선한다.

## 5. CH002 미유

| 범주 | 앵커 |
|---|---|
| 역할 인상 | 말수가 적고 기계에 몰입하는 드론 원거리 딜러 |
| 얼굴 | 성인 여성의 차분하고 무표정한 인상, 장비를 설명할 때만 드러나는 미세한 미소 |
| 헤어 | 라일락·보라 계열 비대칭 단발 `LOCKED`; 한쪽에 무게가 실린 형태 |
| 대표 실루엣 | 큰 작업용 소매와 몸 주변의 부유 드론; 중간 높이의 기술자 실루엣 |
| 장비 | 소형 전투 드론 `LOCKED`; 드론 제어 건틀릿과 공구 키트 |
| 팔레트 | 라일락 / 그레이 / 네온 블루 |
| 대표 특징 | 얼굴·머리 가까이에 읽히는 부유 드론 |
| 보조 특징 | 비대칭 단발, 큰 작업용 소매·건틀릿 |
| `CONTROLLED` | 드론 외피 형태, 소매 길이, 공구 배치, 헤어가 가리는 얼굴 범위 |
| `FORBIDDEN` | 드론보다 큰 장총, 의료 장비, 대형 방패, 루나보다 가벼운 정찰 실루엣 |

드론은 캐릭터와 별개 소품으로 분리 제작하되, 쿼터뷰에서 미유의 머리 주변에 위치해 한
실루엣으로 인식돼야 한다.

## 6. CH003 코코

| 범주 | 앵커 |
|---|---|
| 역할 인상 | 다정함과 응급 상황의 단호함이 공존하는 회복·보호막 서포터 |
| 얼굴 | 성인 여성의 따뜻하고 안정적인 인상, 전투 집중 시 눈매가 단호해지는 대비 |
| 헤어 | 코랄 계열의 부드러운 웨이브 `LOCKED`; 둥근 외곽선과 충분한 헤어 볼륨 |
| 대표 실루엣 | 짧은 의료용 케이프와 둥근 의료 장비; 다섯 명 중 가장 부드러운 곡선 언어 |
| 장비 | 펄스 인젝터 `LOCKED`; 의료 가방과 보호막 발생기 |
| 팔레트 | 코랄 / 아이보리 / 에메랄드 그린 |
| 대표 특징 | 코랄 웨이브 헤어와 의료 케이프 |
| 보조 특징 | 펄스 인젝터, 둥근 보호막 발생기 |
| `CONTROLLED` | 의료 표식 형태, 케이프 길이, 앰플 개수, 보호막 장치 크기 |
| `FORBIDDEN` | 고양이 후드·드론·장총·대형 군용 방패; 공격형 크림슨 중심 팔레트 |

의료 십자나 유사 표식이 필요하면 코코에게만 제한한다. 출시 지역의 심벌 사용 규정을
확인하기 전까지는 독자적인 Re:Camp 의료 픽토그램을 우선한다.

## 7. CH004 이리스

| 범주 | 앵커 |
|---|---|
| 역할 인상 | 완벽한 한 발을 준비하는 냉정한 정밀 사격·보스 딜러 |
| 얼굴 | 성인 여성의 날카롭고 절제된 인상, 칭찬·일상 실수에서만 균열이 보이는 표정 |
| 헤어 | 짙은 퍼플 계열의 길고 정돈된 헤어 `LOCKED`; 세로 흐름이 유지되는 형태 |
| 대표 실루엣 | 긴 비대칭 코트와 긴 에너지 라이플; 삼각형·대각선 중심의 날카로운 언어 |
| 장비 | 에너지 라이플 `LOCKED`; 정밀 조준 바이저 |
| 팔레트 | 퍼플 / 화이트 / 크림슨 |
| 대표 특징 | 키와 비슷한 길이감의 에너지 라이플 |
| 보조 특징 | 조준 바이저, 긴 비대칭 코트 |
| `CONTROLLED` | 포니테일 여부, 코트 분할, 라이플 접힘 구조, 바이저 투명도 |
| `FORBIDDEN` | 짧은 근접 무기 중심 구성, 둥근 의료 실루엣, 대형 방패, 캐주얼 작업복 |

라이플은 화면을 가리지 않도록 접힘·대기·조준 상태를 분리하고, 쿼터뷰에서 총구 방향과
차지 상태가 명확해야 한다.

## 8. CH005 노아

| 범주 | 앵커 |
|---|---|
| 역할 인상 | 가장 앞에서 동료를 지키는 침착한 가드·반격 탱커 |
| 얼굴 | 성인 여성의 침착하고 보호자다운 인상, 무뚝뚝함 안의 따뜻함 |
| 헤어 | 정돈된 짧은 단발 `LOCKED`; 방패와 겹쳐도 머리 외곽선이 남는 형태 |
| 대표 실루엣 | 장신·탄탄한 성인 여성 체형과 허리선이 드러나는 방호 재킷; 사각형·수직선 언어 |
| 장비 | 대형 전개식 에너지 방패 `LOCKED`; 방호 재킷과 유틸리티 모듈 |
| 팔레트 | 네이비 / 스틸 그레이 / 앰버 |
| 대표 특징 | 신체 높이에 가까운 전개식 방패 |
| 보조 특징 | 짧은 단발, 앰버 발광의 사각 모듈 |
| `CONTROLLED` | 방패 접힘 구조, 보호대 면적, 재킷 길이, 체형 근육량 |
| `FORBIDDEN` | 남성적으로 오독되는 얼굴·체형, 장총 중심 구성, 드론·의료 표식, 유아형 치비 |

노아는 여성적 얼굴·허리선과 장신의 힘 있는 체형을 함께 유지한다. 방어구 부피가 얼굴과
체형을 모두 덮어 성별이 모호해지지 않도록 한다.

## 9. 5인 교차 구분 규칙

| 비교 축 | 루나 | 미유 | 코코 | 이리스 | 노아 |
|---|---|---|---|---|---|
| 외곽 형태 | 작고 빠른 곡선 | 소매+부유체 | 둥근 케이프 | 긴 대각선 | 큰 수직 사각형 |
| 헤어 질량 | 짧고 가벼움 | 비대칭 단발 | 풍성한 웨이브 | 길고 정돈됨 | 짧고 단단함 |
| 대표 장비 | 단검 | 드론 | 인젝터·보호막 | 장총 | 대형 방패 |
| 기능 발광 | 민트 | 네온 블루 | 그린 | 크림슨 | 앰버 |
| 전투 자세 | 전방 기울임 | 장비 지시 | 보호·회복 | 후방 조준 | 정면 방어 |

다음 중 하나라도 발생하면 후보를 수정한다.

- 루나와 코코가 의료 표식·케이프 때문에 같은 역할로 보임
- 미유와 이리스가 보라색만으로 구분되고 장비 실루엣이 약함
- 코코와 노아가 모두 원형·대형 방패 사용자처럼 보임
- 다섯 명의 얼굴·체형·키 차이가 사라져 팔레트 스왑처럼 보임
- 대표 장비를 숨기면 쿼터뷰에서 캐릭터를 식별할 수 없음

## 10. 산출물별 앵커 적용

| 산출물 | 반드시 고정 | 검토 포인트 |
|---|---|---|
| 단독 전신 후보 | 얼굴 인상, 헤어 덩어리, 대표 실루엣, 팔레트, 장비 | 성인 여성 매력과 역할 구분 |
| Turnaround | 의상 절개, 장비 결합, 후면 구조, 실제 길이 | 시점 간 구조 일치 |
| 표정 시트 | 얼굴 비율과 주요 특징 | 성격 범위 내 표정 변화 |
| 3D Blockout | 5~6등신, 기본 약 5.2~5.5등신, 머리·손·발·무기 강조, 공용 Rig 호환 | 쿼터뷰 식별성과 관절 가동 |
| Unity Prefab | 재질 슬롯, 장비 소켓, Animator, VFX 소켓 | Gate C와 성능 예산 |
| UI 초상화 | 얼굴·헤어·포인트색 | 작은 크기 식별성 |

이 규칙은 새 도구를 도입하기 위한 조건이 아니다. 현재 도구로도 같은 필드를 기록하고
비교할 수 있으며, 일관성 대량 생성이 실제 병목일 때만 ComfyUI를 재평가한다.
