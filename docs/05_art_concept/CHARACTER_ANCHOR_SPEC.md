# Re:Camp Character Anchor Specification

> Version: 2.0
> 최종 갱신: 2026-07-24
> 상태: Active Design Baseline

이 문서는 `ART-0102`의 도구 비종속 캐릭터 일관성 기준이다. 이미지 생성 도구, Figma,
Blender, Unity가 달라도 루나·미유·코코·이리스·노아를 같은 캐릭터로 유지하기 위해 사용한다.
캐릭터 설정은 `CHARACTER_BIBLE.md`, 공통 표현은 `ART_DIRECTION.md`가 우선한다.

`ReCamp_CharacterLineup_REVIEW_v003.png`는 과거 방향을 점검하는 후보일 뿐이며 승인 원화가 아니다.
사람 승인을 받은 개별 제작 시트가 생기면 해당 시트가 얼굴·의상·장비 구조의 최우선 앵커가 된다.

## 1. 앵커 강도

| 등급 | 의미 | 변경 조건 |
|---|---|---|
| `LOCKED` | 캐릭터 정체성 또는 제품 방향 | 사용자 결정이나 Character Bible 개정 필요 |
| `CONTROLLED` | 같은 캐릭터로 읽히는 범위 안에서 조정 가능 | Review 이력과 변경 이유 기록 |
| `OPEN` | 포즈·표정·세부 배치처럼 제작물별 변형 가능 | 상위 앵커와 충돌하지 않으면 자유 |
| `FORBIDDEN` | 역할 혼동·성별·비율·세계관 회귀 또는 외부 IP 모방 | 제작 후보에서 제외 |

## 2. 전 캐릭터 공통 `LOCKED`

- 플레이어블은 정확히 루나·미유·코코·이리스·노아 5명의 성인 여성이다.
- 2D 대표 일러스트는 약 6.8~7.4등신 범위의 성인 애니메이션 체형이며 기본 목표는 약 7등신이다.
- 인게임 3D는 약 5.2~5.8등신을 중심으로 하는 5~6등신 스타일라이즈드 체형이다.
- 2D와 3D는 같은 캐릭터로 읽혀야 하지만 동일한 신체 비율과 세부 장식 수를 강제하지 않는다.
- 2D는 얼굴·패션·체형·재질을, 3D는 얼굴·헤어·대표 장비·전투 자세와 쿼터뷰 실루엣을 우선한다.
- 3D는 공용 Humanoid 골격과 공용 이동·피격 애니메이션을 사용할 수 있어야 한다.
- 머리·상체·대표 장비 실루엣만으로 Orthographic 쿼터뷰에서 5인을 구분해야 한다.
- 각 캐릭터는 대표 특징 1개와 보조 특징 2개를 우선하고 미세 장식을 늘리지 않는다.
- 캐릭터별 대표색·보조색·포인트색과 역할 장비를 서로 바꾸지 않는다.
- 전원 차콜·웜 그레이·크림 계열의 캠프 공용 재질 언어와 수리 흔적을 일부 공유한다.
- 얼굴 전체를 가리는 헬멧, 2~3등신 유아형 치비, 3D에서 현실적인 6등신 초과의 가늘고 긴 체형,
  과도한 실사·밀리터리·고어를 사용하지 않는다.
- 2D 7등신은 허용·권장하지만 실사 패션 모델처럼 얼굴이 작아져 서브컬처 얼굴 매력이 사라지면 수정한다.
- 외부 작품의 얼굴, 의상 절개, 무기 구조, 로고와 고유 모티프를 직접 복제하지 않는다.
- 생성 결과는 `WIP` 또는 `REVIEW`이며 사람 승인 전 `APPROVED`로 승격하지 않는다.

유행 반영을 이유로 정체성 앵커를 자동 변경하지 않는다. Seed나 모델명처럼 확인할 수 없는 값은 추정하지 않는다.

## 4. CH001 루나

| 범주 | 앵커 |
|---|---|
| 한 문장 훅 | 고양이형 센서 후드와 쌍단검을 쓰는 도시형 폐허 정찰병 |
| 역할 인상 | 빠르고 장난스럽지만 임무 중 판단이 빠른 정찰형 근접 딜러 |
| 2D 비율 | 약 6.9~7.1등신 `LOCKED` |
| 3D 비율 | 약 5.3~5.4등신 `LOCKED` |

| 헤어 | 민트가 섞인 은회색 울프 보브 또는 가벼운 중단발, 한쪽 비대칭 포인트 |
| 체형 | 슬림 애슬레틱, 민첩한 하체와 낮은 준비 자세 |
| 대표 실루엣 | 고양이형 센서 후드 `LOCKED`; 짧은 재킷, 비대칭 허리 패널, 쌍단검 |
| 의상 | 차콜 기능성 이너 + 크림 크롭 정찰 재킷 + 하이웨이스트 유틸리티 쇼츠 |
| 장비 | 쌍 에너지 단검 `LOCKED`; 손목 스캐너와 소형 백팩 |
| 팔레트 | 민트 실버 / 차콜 / 크림 / 핫 핑크 소량; 청록 발광 |
| 대표 특징 | 접힘·회전이 가능한 고양이형 센서 귀 |
| 보조 특징 | 쌍단검, 꼬리처럼 보이는 스캐너 케이블 |
| `CONTROLLED` | 재킷 길이, 쇼츠와 스커트 패널 비율, 헤어 끝 형태, 노출도, 단검 날 구조 |
| `FORBIDDEN` | 생물 귀 중심 수인화, 의료 표식, 장총, 대형 방패, 보라색 암살자 중심 회귀 |

후드와 귀는 장식이 아니라 탐지 장비여야 한다. 귀, 얼굴, 쌍단검 중 두 개 이상이 숨겨지면 루나로
인식하기 어려우므로 후보를 수정한다.

| `FORBIDDEN` | 드론보다 큰 장총, 양쪽 대칭 소매, 의료 장비, 대형 방패, 이리스와 같은 긴 보라 헤어 |

드론은 별도 소품으로 분리 제작하지만 전투 카메라에서는 미유의 머리 주변에 위치해 한 캐릭터의
실루엣으로 인식돼야 한다.

곡선적인 VFX여야 한다.

이리스의 핵심 차별점이므로 전체가 어두운 전술복으로 회귀하면 수정한다.

| `FORBIDDEN` | 얼굴·허리선을 덮는 중장갑, 남성적으로 오독되는 얼굴, 장총, 드론·의료 표식, 원형 힐러 방패 |

노아의 힘은 갑옷 부피가 아니라 체형, 자세와 방패 전개 연출에서 보여준다. 방패가 접혔을 때도
세로 케이스 형태가 남아야 한다.

| 체형 | 슬림 애슬레틱 | 작고 마른 성인 | 건강한 곡선 | 우아한 장신 | 탄탄한 최장신 |
| 대표 의상 | 크림 크롭 재킷 | 한쪽 큰 워크 보머 | 구조복·하프 케이프 | 화이트 롱 코트 | 테일러드 방호 재킷 |
| 대표 장비 | 쌍단검·센서 귀 | 드론 2기 | 인젝터·투명 보호막 | 장거리 라이플 | 전개식 대형 방패 |
| 기능 발광 | 청록 | 네온 블루 | 에메랄드 | 크림슨 | 앰버 |
| 전투 자세 | 전방 기울임 | 장비 지시 | 구조·보호 | 후방 정렬 조준 | 정면 방어 |
| 감정 대비 | 장난↔집중 | 무표정↔기계 열정 | 미소↔현장 지휘 | 완벽↔생활 허당 | 무뚝뚝↔행동형 배려 |

다음 중 하나라도 발생하면 후보를 수정한다.

- 루나가 일반적인 은발·보라색 총기 요원으로 보임
- 미유와 이리스가 보라색 헤어만으로 구분되고 드론·화이트 코트 차이가 약함
- 코코가 간호사 코스튬 또는 노아와 같은 물리 방패 사용자로 보임
- 이리스가 흔한 검정 암살자나 루나의 민첩 캐릭터처럼 보임
- 노아가 전신 중장갑 남성 탱커처럼 보이거나 코코의 보호막과 구분되지 않음
- 다섯 명의 얼굴·체형·키 차이가 사라져 팔레트 스왑처럼 보임
- 대표 장비를 숨겼을 때 헤어·의상 외곽선으로 5인을 구분할 수 없음

## 10. 2D 7등신 → 3D 5~6등신 번역 규칙

| 항목 | 2D 대표 일러스트 | 3D 인게임 번역 |
|---|---|---|
| 비율 | 약 6.8~7.4등신, 긴 다리와 패션 실루엣 | 약 5.2~5.8등신, 머리·손·발·장비 확대 |
| 얼굴 | 눈·코·입·턱과 미세 표정 표현 | 눈매·헤어라인·대표 표정 3~5종 우선 |
| 의상 | 레이어와 절개·재질을 풍부하게 표현 | 큰 색면 3~5개와 실루엣을 남기고 미세 장식 축소 |
| 긴 천·헤어 | 키 아트의 흐름과 우아함 강조 | 길이를 줄이고 물리 본·관통 예산에 맞춤 |
| 장비 | 구조와 사용법을 상세하게 설명 | 화면에서 128px 이하로 보아도 종류와 방향이 읽히도록 확대 |
| 색상 | 미묘한 색 변화와 재질 차이 허용 | 배경과 분리되는 명도·채도 대비 우선 |
| 노출도 | 캐릭터 매력과 패션 맥락에서 결정 | 카메라·포즈·관절 변형과 불필요한 시선 분산 검토 |

3D는 2D의 축소판이 아니라 플레이용 번역본이다. 사람 승인된 제작 시트에서 무엇을 확대·삭제·결합했는지
`2D to 3D Simplification` 필드에 기록한다.

## 11. 산출물별 앵커 적용

| 산출물 | 반드시 고정 | 검토 포인트 |
|---|---|---|
| 2D 단독 전신 후보 | 약 7등신 범위, 얼굴 인상, 헤어 덩어리, 패션 실루엣, 팔레트, 장비 | 성인 여성 매력과 한 문장 훅 |
| Turnaround | 의상 절개, 장비 결합, 후면 구조, 실제 길이 | 시점 간 구조 일치와 3D 제작 가능성 |
| 표정 시트 | 얼굴 비율과 주요 특징 | 성격의 기본·반전 상태가 모두 표현되는지 |
| 3D Blockout | 캐릭터별 목표 등신, 머리·손·발·무기 강조, 공용 Rig 호환 | 쿼터뷰 식별성과 관절 가동 |
| Unity Prefab | 재질 슬롯, 장비 소켓, Animator, VFX 소켓 | Gate C와 성능 예산 |
| UI 초상화 | 얼굴·헤어·포인트색 | 작은 크기 식별성과 감정 전달 |
| 스토어 Key Art | 7등신 체형, 대표 표정·포즈·장비 | 캐릭터 훅과 게임 세계가 한 장에서 전달되는지 |

이 규칙은 특정 이미지 생성 도구를 도입하기 위한 조건이 아니다. 현재 도구로도 같은 필드를 기록하고
비교할 수 있으며, 일관성 대량 생성이 실제 병목일 때만 ComfyUI를 재평가한다.
