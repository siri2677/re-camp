# Re:Camp Art Direction

## 1. 최종 비주얼 정의

Re:Camp의 비주얼은 **매력적인 성인 여성 서브컬쳐 캐릭터 + 자연에 잠식된 부드러운 포스트 아포칼립스 + 희망적인 캠프 생활**의 조합이다. 핵심 타깃은 미소녀 캐릭터의 수집·성장·애착을 중시하는 남성향 서브컬쳐 RPG 이용자다.

```text
2D 일반 등신 캐릭터 일러스트
+ 3D 4~5등신 SD-lite 스타일라이즈드 인게임 캐릭터
+ 3D 스타일라이즈드 폐허와 캠프
+ 툰 셰이딩
+ 고정 쿼터뷰 전투
```

배경의 사실성보다 캐릭터의 매력, 식별성, 애착 형성을 우선한다.

## 2. 표현 방식

| 구분 | 최종 방향 |
|---|---|
| 캐릭터 대표 이미지 | 2D 일반 등신 서브컬쳐 일러스트 |
| 인게임 캐릭터 | 원본 전투 콘셉트 계열의 3D 4~5등신 SD-lite 스타일라이즈드 모델 |
| 카메라 | 고정 Orthographic 쿼터뷰 |
| 렌더링 | 툰 셰이딩 기반 3D |
| 배경 | 자연에 잠식된 스타일라이즈드 폐허 |
| UI | 둥근 패널, 가벼운 글래스, PC·모바일 가독성 |
| 분위기 | 위협적인 세계와 따뜻한 캐릭터·캠프의 대비 |

## 3. 캐릭터 공통 디자인 규칙

- 플레이어블 5명은 모두 성인 여성이다. 남성 플레이어블 캐릭터는 현재 출시 범위에 포함하지 않는다.
- 미소녀·여성 캐릭터의 얼굴, 체형, 헤어, 의상 매력을 최우선으로 하는 남성향 서브컬쳐 디자인을 사용한다.
- 여성적인 실루엣을 유지하되 역할과 성격에 따라 체형·키·분위기를 구분한다.
- 가벼운 택티컬 스트리트 패션을 기반으로 캐릭터별 노출도와 장식은 매력·전투 가독성·3D 제작 가능성을 함께 검토한다.
- 후드, 고글, 백팩, 드론, 의료·정밀 장비 같은 기능성 소품을 활용한다.
- 대표색 1개, 보조색 1개, 포인트색 1개를 사용한다.
- 작은 화면에서도 머리, 상체, 무기 실루엣으로 구분되어야 한다.
- 원본 전투 콘셉트처럼 머리·손·무기를 과장한 4~5등신 SD-lite 3D를 사용한다. 2~3등신 유아형 비율은 피한다.
- 공용 Humanoid 골격과 공용 이동·피격 애니메이션을 사용한다.
- 얼굴, 헤어, 의상, 무기, 대표 모션은 개별 제작한다.
- 기억에 남는 대표 특징 1개와 보조 특징 2개를 복잡한 장식보다 우선한다.

| CH002 | 미유 | 드론 원거리 딜러 | 전투 드론, 제어 건틀릿 | 라일락 / 그레이 / 네온 블루 |

## 5. 캐릭터별 실루엣 원칙

### 루나

- 고양이 귀 후드, 짧은 재킷, 활동적인 하체 실루엣
- 에너지 단검과 소형 탐지 장비
- 가볍고 빠른 인상

### 미유

- 비대칭 단발, 큰 작업용 소매
- 머리 주변의 부유 드론과 공구·제어 장치
- 기술자다운 라일락·네온 컬러

### 코코

- 부드러운 웨이브 헤어와 의료용 케이프
- 둥근 보호막 발생기와 의료 가방
- 따뜻하고 안정적인 곡선 실루엣

### 이리스

- 길고 정돈된 헤어와 비대칭 코트
- 긴 에너지 라이플
- 날카로운 삼각형 중심의 정밀한 실루엣

### 노아

- 장신의 탄탄한 성인 여성 수호자
- 정돈된 단발과 허리선이 드러나는 방호 재킷
- 대형 전개식 방패와 사각형·수직선 중심의 안정적인 실루엣

## 6. 세계와 환경

### 배경 테마

- 폐허 거리
- 오염된 숲
- 버려진 연구소
- 지하 상가
- 기계 무덤
- 재건 중인 생존자 캠프

배경은 잔혹하고 절망적인 폐허보다 **시간과 자연에 의해 아름답게 남겨진 공간**을 지향한다. 캐릭터보다 디테일과 채도를 낮추고 이동, 위험, 자원, 상호작용 영역을 명확히 구분한다.

### 캠프

캠프는 전투 지역과 시각적으로 대비되는 안전한 장소다.

- 따뜻한 조명
- 천막, 안테나, 발전기, 작업대, 식량 저장고
- 캐릭터별 개인 공간과 생활 소품
- 수리 흔적이 보이지만 정돈된 구조
- 성장에 따라 증가하는 조명과 생활감

## 7. 컬러 언어

```text
프로젝트 공통: 민트 / 크림 / 라이트 퍼플
희망과 회복: 코랄 / 핑크 / 앰버
기술 요소: 네온 블루 / 퍼플
위험 요소: 크림슨 / 딥 퍼플
배경: 그레이시 블루 / 파스텔 그린 / 웜 그레이
```

캐릭터와 공격 VFX는 배경보다 높은 명도·채도 대비를 가진다.

## 8. 재질과 셰이딩

- 캐릭터: 깨끗한 툰 셰이딩, 얼굴 그림자 최소화
- 의상: 천, 합성 섬유, 무광 플라스틱
- 장비: 마모된 금속과 제한적인 기능성 발광 패널
- 배경: 캐릭터보다 낮은 디테일과 채도
- 발광: 기능이 있는 장치와 전투 피드백에만 제한적으로 사용

## 9. UI와 VFX

### UI

- 둥근 버튼과 패널, 얇은 외곽선과 반투명 글래스
- PC·모바일에서 읽히는 글자와 충분한 터치 영역
- 전투 HUD 최소화
- 캐릭터별 포인트 컬러를 선택 화면과 스킬 UI에 반영

### VFX

- 루나: 민트·청록의 짧고 날카로운 궤적
- 미유: 라일락·네온 블루 홀로그램과 드론 탄도
- 코코: 코랄·그린 회복 파동과 보호막
- 이리스: 퍼플·크림슨 조준선과 압축 타격광
- 노아: 네이비·앰버 방벽과 반격 충격파

## 10. 금지 스타일

- 과도한 실사·무거운 AAA 밀리터리·잔혹한 고어
- 얼굴을 가리는 과도한 장비
- 모바일·쿼터뷰에서 읽히지 않는 미세 장식
- 캐릭터마다 다른 세계관처럼 보이는 디자인
- 남성 플레이어블 캐릭터 또는 성별이 모호하게 읽히는 최종 디자인
- 2~3등신 유아형 치비 또는 여성 캐릭터의 얼굴·체형 매력을 읽기 어려운 과도한 데포르메
- AI 결과를 구조 검수 없이 최종 원화·3D 설계도로 사용

## 11. 레퍼런스 상태와 우선순위

- 생성 이미지는 WIP 또는 REVIEW이며 자동으로 APPROVED되지 않는다.
- 승인된 개별 캐릭터 제작 시트가 라인업·무드보드보다 우선한다.
- `battle_vertical_slice_concept.png`가 기존에 합의한 원본 전투 콘셉트다. 화면 구도·카메라·폐허 분위기·색 대비의 1차 시각 기준으로 유지한다.
- 원본 전투 콘셉트의 캐릭터는 여성 주인공의 분위기와 화면 내 존재감 참고에 사용하되, 정확한 체형·의상·무기 구조는 개별 제작 시트에서 확정한다.
- 원격 라인업 v001은 시각적으로 불량해 사용하지 않는다.
- `ReCamp_CharacterLineup_REJECTED_v002.png`는 전원 여성 조건 누락과 원본 계보 오인으로 반려됐으며 후속 제작에 사용하지 않는다.
- `ReCamp_CharacterLineup_REVIEW_v003.png`는 성인 여성 5인·4~5등신 SD-lite 계열은 유효하지만
  `ReCamp_CharacterLineup_REVIEW_v003.review.md`에서 Gate A `FAIL` 권고를 받았다. 역할 모티프,
  얼굴, 실루엣을 캐릭터별 WIP에서 교정하고 사람이 재검토하기 전까지 제작 승인본이 아니다.
