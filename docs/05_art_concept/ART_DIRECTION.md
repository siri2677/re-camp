# Re:Camp Art Direction

## 최종 비주얼 정의

Re:Camp의 비주얼은 매력적인 성인 여성 서브컬쳐 캐릭터, 자연에 잠식된 부드러운 포스트 아포칼립스, 희망적인 캠프 생활의 조합이다. 핵심 타깃은 미소녀 캐릭터의 수집·성장·애착을 중시하는 남성향 서브컬쳐 RPG 이용자다.

```text
2D 일반 등신 캐릭터 일러스트
+ 3D 4~5등신 SD-lite 스타일라이즈드 인게임 캐릭터
+ 3D 스타일라이즈드 폐허와 캠프
+ 툰 셰이딩
+ 고정 쿼터뷰 전투
```

배경의 사실성보다 캐릭터의 매력, 식별성, 애착 형성을 우선한다.

## 표현 방식

| 구분 | 방향 |
|---|---|
| 캐릭터 대표 이미지 | 2D 일반 등신 서브컬쳐 일러스트 |
| 인게임 캐릭터 | 원본 전투 콘셉트 계열의 3D 4~5등신 SD-lite 스타일라이즈드 모델 |
| 카메라 | 고정 Orthographic 쿼터뷰 |
| 렌더링 | 툰 셰이딩 기반 3D |
| 배경 | 자연에 잠식된 스타일라이즈드 폐허 |
| UI | 둥근 패널, 가벼운 글래스, PC·모바일 가독성 |
| 분위기 | 위협적인 세계와 따뜻한 캐릭터·캠프의 대비 |

## 캐릭터 공통 규칙

- 플레이어블 5명은 모두 성인 여성이다. 남성 플레이어블 캐릭터는 현재 출시 범위에 포함하지 않는다.
- 미소녀·여성 캐릭터의 얼굴, 체형, 헤어, 의상 매력을 최우선으로 하는 남성향 서브컬쳐 디자인
- 여성적인 실루엣을 유지하되 역할과 성격에 따라 체형·키·분위기를 구분한다.
- 가벼운 택티컬 스트리트 패션을 기반으로 캐릭터별 노출도와 장식은 매력·전투 가독성·3D 제작 가능성을 함께 검토한다.
- 후드, 고글, 백팩, 드론, 의료·정밀 장비 같은 기능성 소품
- 대표색 1개, 보조색 1개, 포인트색 1개
- 작은 화면에서도 머리, 상체, 무기 실루엣으로 구분
- 원본 전투 콘셉트처럼 머리·손·무기를 과장한 4~5등신 SD-lite 3D. 2~3등신 유아형 비율은 피한다.
- 공용 Humanoid 골격, 공용 이동·피격 애니메이션
- 얼굴, 헤어, 의상, 무기, 대표 모션은 개별 제작
- 기억에 남는 대표 특징 1개와 보조 특징 2개를 복잡한 장식보다 우선

## 플레이어블 라인업

| ID | 캐릭터 | 역할 | 무기·장비 | 대표 색상 |
|---|---|---|---|---|
| CH001 | 루나 | 정찰형 근접 딜러 | 에너지 단검, 탐지 장치 | 민트 / 크림 / 핑크 |
| CH002 | 미유 | 드론 원거리 딜러 | 전투 드론, 제어 건틀릿 | 라일락 / 그레이 / 네온 블루 |
| CH003 | 코코 | 회복·보호막 서포터 | 펄스 인젝터, 의료 장비 | 코랄 / 아이보리 / 그린 |
| CH004 | 이리스 | 정밀 사격·보스 딜러 | 에너지 라이플, 조준 장치 | 퍼플 / 화이트 / 크림슨 |
| CH005 | 노아 | 가드·반격 탱커 | 전개식 에너지 방패 | 네이비 / 스틸 / 앰버 |

세부 설정은 `CHARACTER_BIBLE.md`를 단일 기준으로 한다.

## 실루엣

- 루나: 고양이 귀 후드, 짧은 재킷, 에너지 단검, 가볍고 빠른 인상
- 미유: 비대칭 단발, 큰 작업 소매, 머리 주변 부유 드론
- 코코: 웨이브 헤어, 의료 케이프, 둥근 보호막·의료 장비
- 이리스: 긴 정돈 헤어, 비대칭 코트, 긴 에너지 라이플
- 노아: 장신의 성인 여성 수호자, 정돈된 단발, 허리선이 드러나는 방호 재킷, 대형 전개식 방패, 안정적인 실루엣

## 세계와 환경

- 폐허 거리, 오염된 숲, 버려진 연구소, 지하 상가, 기계 무덤
- 잔혹하고 절망적인 공간보다 시간과 자연에 의해 아름답게 남은 폐허
- 캐릭터보다 낮은 디테일·채도
- 이동, 위험, 자원, 상호작용 영역의 명확한 구분

캠프는 따뜻한 조명, 천막, 안테나, 발전기, 작업대, 식량 저장고와 캐릭터별 생활 소품이 있는 안전 지대다. 성장할수록 조명과 생활감이 증가한다.

## 컬러 언어

```text
프로젝트 공통: 민트 / 크림 / 라이트 퍼플
희망과 회복: 코랄 / 핑크 / 앰버
기술: 네온 블루 / 퍼플
위험: 크림슨 / 딥 퍼플
배경: 그레이시 블루 / 파스텔 그린 / 웜 그레이
```

캐릭터와 공격 VFX는 배경보다 높은 명도·채도 대비를 가진다.

## 재질·UI·VFX

- 캐릭터: 깨끗한 툰 셰이딩, 얼굴 그림자 최소화
- 의상: 천, 합성 섬유, 무광 플라스틱
- 장비: 마모된 금속과 제한적인 기능성 발광 패널
- UI: 둥근 버튼·패널, 얇은 외곽선, 반투명 글래스, 최소 전투 HUD
- 루나 VFX: 민트·청록의 짧고 날카로운 궤적
- 미유 VFX: 라일락·네온 블루 홀로그램과 드론 탄도
- 코코 VFX: 코랄·그린 회복 파동과 보호막
- 이리스 VFX: 퍼플·크림슨 조준선과 압축 타격광
- 노아 VFX: 네이비·앰버 방벽과 반격 충격파

## 금지 스타일

- 과도한 실사·무거운 AAA 밀리터리·잔혹한 고어
- 얼굴을 가리는 과도한 장비
- 모바일·쿼터뷰에서 읽히지 않는 미세 장식
- 캐릭터마다 다른 세계관처럼 보이는 디자인
- 남성 플레이어블 캐릭터 또는 성별이 모호하게 읽히는 최종 디자인
- 2~3등신 유아형 치비 또는 여성 캐릭터의 얼굴·체형 매력을 읽기 어려운 과도한 데포르메
- AI 결과를 구조 검수 없이 최종 원화·3D 설계도로 사용

## 레퍼런스 상태

- 생성 이미지는 WIP 또는 REVIEW이며 자동으로 Approved되지 않는다.
- 승인된 개별 캐릭터 제작 시트가 라인업·무드보드보다 우선한다.
- 원격 라인업 v001은 시각적으로 불량해 사용하지 않는다.
- `battle_vertical_slice_concept.png`가 기존에 합의한 원본 전투 콘셉트다. 화면 구도·카메라·폐허 분위기·색 대비의 1차 시각 기준으로 유지한다.
- 원본 전투 콘셉트의 캐릭터는 여성 주인공의 분위기와 화면 내 존재감 참고에 사용하되, 정확한 체형·의상·무기 구조는 개별 제작 시트에서 확정한다.
- `ReCamp_CharacterLineup_REJECTED_v002.png`는 전원 여성 조건 누락과 원본 계보 오인으로 반려됐으며 후속 제작에 사용하지 않는다.
