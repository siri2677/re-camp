# Re:Camp Art Direction

## 1. 최종 비주얼 정의

Re:Camp의 비주얼 방향은 **매력적인 서브컬쳐 캐릭터 + 자연에 잠식된 부드러운 포스트 아포칼립스 + 희망적인 캠프 생활**이다.

```text
2D 일반 등신 캐릭터 일러스트
+ 3D 5~6등신 스타일라이즈드 인게임 캐릭터
+ 3D 스타일라이즈드 폐허와 캠프
+ URP 툰 셰이딩
+ 고정 쿼터뷰 전투
```

아트의 최우선 목표는 배경의 사실성이 아니라 캐릭터의 매력, 식별성, 애착 형성이다.

## 2. 표현 방식

| 구분 | 최종 방향 |
|---|---|
| 캐릭터 대표 이미지 | 2D 일반 등신 서브컬쳐 일러스트 |
| 인게임 캐릭터 | 3D 5~6등신 스타일라이즈드 모델 |
| 카메라 | 고정 쿼터뷰 |
| 렌더링 | URP 툰 셰이딩 기반 3D |
| 배경 | 자연에 잠식된 스타일라이즈드 폐허 |
| UI | PC 우선 가독성, Android Touch 확장 가능 구조 |
| 분위기 | 위협적인 세계와 따뜻한 캐릭터·캠프의 대비 |

## 3. 캐릭터 공통 디자인 규칙

- 미소녀·귀여운 캐릭터 중심
- 가벼운 택티컬 스트리트 패션
- 후드, 고글, 백팩, 드론, 의료 장비, 정밀 장비 등 기능성 소품 활용
- 대표 색상 1개, 보조 색상 1개, 포인트 색상 1개 사용
- 화면에서 작게 보여도 머리·상체·무기 실루엣으로 구분 가능해야 함
- 7~8등신 실사 비율보다 머리·손·발·무기를 조금 강조한 5~6등신 사용
- 공용 Humanoid 골격을 사용하되 얼굴·헤어·의상·무기·대표 모션은 개별 제작
- 복잡한 장식보다 기억에 남는 대표 특징 1개와 보조 특징 2개 우선
- 쿼터뷰에서 보이지 않거나 물리 비용만 높은 장식은 제거·확대·Texture화

## 4. 플레이어블 캐릭터 라인업

| ID | 캐릭터 | 역할 | 무기·장비 | 대표 색상 |
|---|---|---|---|---|
| CH001 | 루나 | 정찰형 근접 딜러 | 에너지 단검, 탐지 장치 | 민트 / 크림 / 핑크 |
| CH002 | 미유 | 드론 원거리 딜러 | 전투 드론, 제어 건틀릿 | 라일락 / 그레이 / 네온 블루 |
| CH003 | 코코 | 회복·보호막 서포터 | 펄스 인젝터, 의료 장비 | 코랄 / 아이보리 / 그린 |
| CH004 | 이리스 | 정밀 사격·보스 딜러 | 에너지 라이플, 조준 장치 | 퍼플 / 화이트 / 크림슨 |
| CH005 | 노아 | 가드·반격 탱커 | 전개식 에너지 방패 | 네이비 / 스틸 / 앰버 |

세부 설정의 단일 기준은 `CHARACTER_BIBLE.md`다.

현재 통합 라인업은 `Direction Approved` 상태다.

- 역할·대표 색상·무기·큰 실루엣은 유지 가능
- 개별 얼굴·헤어·의상·Turnaround는 아직 최종 승인 전
- 개별 `APPROVED` 제작 시트가 완성되면 해당 시트를 최우선 사용

## 5. 캐릭터별 실루엣

### 루나

- 고양이 귀 후드
- 짧은 재킷과 활동적인 하체
- 에너지 단검과 소형 탐지 장비
- 빠르고 가벼운 인상

### 미유

- 비대칭 단발과 큰 작업용 소매
- 머리 주변 부유 드론
- 공구와 제어 장치가 밀집된 실루엣

### 코코

- 부드러운 웨이브 헤어와 의료 케이프
- 둥근 보호막 발생기와 의료 가방
- 따뜻하고 안정적인 곡선

### 이리스

- 길고 정돈된 헤어와 비대칭 코트
- 긴 에너지 라이플
- 삼각형과 직선 중심의 정밀한 실루엣

### 노아

- 넓은 어깨의 방호 재킷
- 대형 전개식 방패
- 사각형과 수직선 중심의 안정적인 실루엣

## 6. 세계와 환경

### 핵심 지역

- 버려진 거리
- 재건 중인 생존자 캠프
- 폐쇄된 연구소

장기 확장 후보:

- 오염된 숲
- 지하 상가
- 기계 무덤

배경은 잔혹하고 절망적인 폐허보다 **시간과 자연에 의해 아름답게 남겨진 공간**을 지향한다.

### 캠프

- 따뜻한 조명
- 천막·안테나·발전기·작업대·식량 창고
- 수리 흔적이 있지만 정돈된 구조
- 성장에 따라 조명·시설·생활감 증가
- 전투 지역과 명확히 대비되는 안전한 공간

## 7. 컬러 언어

```text
프로젝트 공통: 민트 / 크림 / 라이트 퍼플
희망·회복: 코랄 / 핑크 / 앰버
기술: 네온 블루 / 퍼플
위험: 크림슨 / 딥 퍼플 / 주황
배경: 그레이시 블루 / 파스텔 그린 / 웜 그레이
```

캐릭터와 공격 VFX는 배경보다 높은 명도·채도 대비를 갖는다.

## 8. 재질과 셰이딩

- 캐릭터: 깨끗한 1~2단 툰 셰이딩, 얼굴 그림자 최소화
- 의상: 천·합성 섬유·무광 플라스틱
- 장비: 마모된 금속과 제한적 발광 패널
- 배경: 캐릭터보다 낮은 디테일과 채도
- 발광: 기능이 있는 장치와 전투 피드백에만 사용
- Outline과 Rim Light: 배경 분리에 필요한 정도로 제한

## 9. UI

```text
크림색·저채도 Panel
+ 라벤더·민트 Accent
+ 짙은 남색 Text
+ 둥근 Button과 Card
+ 제한적 Glass 표현
```

- Windows 1920×1080 기준으로 먼저 검수
- Keyboard/Mouse·Gamepad Focus 상태 제공
- Android 확장 시 Safe Area·Touch Target·HUD Scale 적용
- PC 화면에서 모바일 버튼처럼 과도하게 크게 만들지 않음
- 전투 중심부를 가리는 대형 Panel 금지

## 10. VFX

- 루나: 민트·청록의 짧고 날카로운 궤적
- 미유: 라일락·네온 블루의 홀로그램과 드론 탄도
- 코코: 코랄·그린의 회복 파동과 보호막
- 이리스: 퍼플·크림슨의 정밀 조준선과 압축된 타격광
- 노아: 네이비·앰버의 방벽과 반격 충격파

적 위험 표시는 빨강·주황과 Telegraph 형태로 구분하며 캐릭터 VFX와 혼동되지 않아야 한다.

## 11. 금지 스타일

- 과도한 실사풍과 무거운 AAA 밀리터리 표현
- 잔혹한 고어와 지속적으로 어두운 화면
- 얼굴을 가리는 과도한 장비
- 쿼터뷰에서 식별되지 않는 미세 장식
- SD·치비로 보이는 3~4등신 인게임 캐릭터
- 캐릭터마다 서로 다른 세계관처럼 보이는 디자인
- AI 생성 결과를 구조 검수 없이 최종 원화·Turnaround·3D 설계도로 사용
- 가챠·상점 중심 게임처럼 보이는 불필요한 별 등급·결제 UI

## 12. 제작 우선순위

```text
5인 방향 유지
→ 루나 Approved 제작 시트
→ 루나 5~6등신 Blockout
→ Coplay/Aura Unity 검증
→ 루나 Character Proof
→ Vertical Slice
→ 나머지 캐릭터 확장
```

루나 Proof 전 나머지 네 캐릭터의 고비용 최종 모델링을 진행하지 않는다.
