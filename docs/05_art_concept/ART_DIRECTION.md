# Re:Camp Art Direction

## 1. 최종 비주얼 정의

Re:Camp의 비주얼 방향은 **매력적인 서브컬쳐 캐릭터 + 자연에 잠식된 부드러운 포스트 아포칼립스 + 희망적인 캠프 생활**이다.

```text
2D 일반 등신 캐릭터 일러스트
+ 3D 5~6등신 스타일라이즈드 인게임 캐릭터
+ 3D 스타일라이즈드 폐허와 캠프
+ 툰 셰이딩
+ 고정 쿼터뷰 전투
```

아트의 최우선 목표는 배경의 사실성이 아니라 **캐릭터의 매력, 식별성, 애착 형성**이다.

## 2. 표현 방식

| 구분 | 최종 방향 |
|---|---|
| 캐릭터 대표 이미지 | 2D 일반 등신 서브컬쳐 일러스트 |
| 인게임 캐릭터 | 3D 5~6등신 스타일라이즈드 모델 |
| 카메라 | 고정 쿼터뷰 |
| 렌더링 | 툰 셰이딩 기반 3D |
| 배경 | 자연에 잠식된 스타일라이즈드 폐허 |
| UI | 둥근 패널, 가벼운 글래스 표현, 모바일 가독성 우선 |
| 분위기 | 위협적인 세계와 따뜻한 캐릭터·캠프의 대비 |

## 3. 캐릭터 공통 디자인 규칙

- 미소녀·귀여운 캐릭터 중심
- 가벼운 택티컬 스트리트 패션
- 후드, 고글, 백팩, 드론, 의료 장비, 정밀 장비 등 기능성 소품 활용
- 캐릭터마다 대표 색상 1개, 보조 색상 1개, 포인트 색상 1개 사용
- 화면에서 작게 보여도 머리, 상체, 무기 실루엣으로 구분 가능해야 함
- 7~8등신 실사 비율보다 머리·손·무기를 조금 과장한 5~6등신 사용
- 모든 캐릭터는 공용 Humanoid 골격을 사용하되 얼굴, 헤어, 의상, 무기, 대표 모션은 개별 제작
- 복잡한 장식 수보다 기억에 남는 대표 특징 1개와 보조 특징 2개를 우선

## 4. 플레이어블 캐릭터 라인업

| ID | 캐릭터 | 역할 | 무기·장비 | 대표 색상 |
|---|---|---|---|---|
| CH001 | 루나 | 정찰형 근접 딜러 | 에너지 단검, 탐지 장치 | 민트 / 크림 / 핑크 |
| CH002 | 미유 | 드론 원거리 딜러 | 전투 드론, 제어 건틀릿 | 라일락 / 그레이 / 네온 블루 |
| CH003 | 코코 | 회복·보호막 서포터 | 펄스 인젝터, 의료 장비 | 코랄 / 아이보리 / 그린 |
| CH004 | 이리스 | 정밀 사격·보스 딜러 | 에너지 라이플, 조준 장치 | 퍼플 / 화이트 / 크림슨 |
| CH005 | 노아 | 가드·반격 탱커 | 전개식 에너지 방패 | 네이비 / 스틸 / 앰버 |

## 5. 캐릭터별 실루엣 원칙

### 루나

- 고양이 귀 후드
- 짧은 재킷과 활동적인 하체 실루엣
- 에너지 단검과 소형 탐지 장비
- 빠르고 가벼운 인상

### 미유

- 비대칭 단발과 큰 작업용 소매
- 머리 주변의 부유 드론
- 공구와 제어 장치가 밀집된 실루엣
- 기술자다운 라일락·네온 컬러

### 코코

- 부드러운 웨이브 헤어와 의료용 케이프
- 둥근 보호막 발생기와 의료 가방
- 따뜻하고 안정적인 곡선 실루엣

### 이리스

- 길고 정돈된 헤어와 비대칭 코트
- 긴 에너지 라이플
- 날카로운 삼각형 중심의 정밀한 실루엣

### 노아

- 넓은 어깨의 방호 재킷
- 대형 전개식 방패
- 사각형과 수직선 중심의 안정적인 실루엣

## 6. 세계와 환경

### 배경 테마

- 폐허 거리
- 오염된 숲
- 버려진 연구소
- 지하 상가
- 기계 무덤
- 재건 중인 생존자 캠프

배경은 잔혹하고 절망적인 폐허보다 **시간과 자연에 의해 아름답게 남겨진 공간**을 지향한다.

### 캠프

캠프는 전투 지역과 시각적으로 대비되는 안전한 장소다.

- 따뜻한 조명
- 천막, 안테나, 발전기, 작업대, 식량 저장고
- 캐릭터별 개인 공간과 생활 소품
- 수리 흔적이 보이지만 정돈된 구조
- 성장에 따라 조명과 생활감이 증가

## 7. 컬러 언어

```text
프로젝트 공통: 민트 / 크림 / 라이트 퍼플
희망과 회복: 코랄 / 핑크 / 앰버
기술 요소: 네온 블루 / 퍼플
위험 요소: 크림슨 / 딥 퍼플
배경: 그레이시 블루 / 파스텔 그린 / 웜 그레이
```

캐릭터와 공격 이펙트의 색이 배경보다 높은 명도·채도 대비를 가져야 한다.

## 8. 재질과 셰이딩

- 캐릭터: 깨끗한 툰 셰이딩, 얼굴 그림자 최소화
- 의상: 천, 합성 섬유, 무광 플라스틱 중심
- 장비: 마모된 금속과 부드러운 발광 패널
- 배경: 캐릭터보다 낮은 디테일과 채도
- 발광: 기능이 있는 장치와 전투 피드백에만 제한적으로 사용

## 9. UI와 VFX

### UI

- 둥근 버튼과 패널
- 얇은 외곽선과 반투명 글래스 표현
- 모바일 터치 영역 우선
- 전투 HUD 최소화
- 캐릭터별 포인트 컬러를 선택 화면과 스킬 UI에 반영

### VFX

- 루나: 민트·청록의 짧고 날카로운 궤적
- 미유: 라일락·네온 블루의 홀로그램과 드론 탄도
- 코코: 코랄·그린의 부드러운 회복 파동과 보호막
- 이리스: 퍼플·크림슨의 정밀 조준선과 압축된 타격광
- 노아: 네이비·앰버의 육중한 방벽과 반격 충격파

## 10. 금지 스타일

- 과도한 실사풍과 무거운 AAA 밀리터리 표현
- 잔혹한 고어와 지속적으로 어두운 화면
- 캐릭터 얼굴을 가리는 과도한 장비
- 모바일 화면에서 식별되지 않는 미세 장식
- 캐릭터마다 서로 다른 세계관처럼 보이는 디자인
- AI 생성 결과를 구조 검수 없이 최종 원화나 3D 설계도로 사용하는 것

## 11. 현재 레퍼런스 상태

`art_refs/art_direction/`의 이미지는 **통합 1차 콘셉트 레퍼런스**다.

- 캐릭터 라인업과 색상·역할 통일 기준으로 사용
- 최종 원화, Turnaround, 3D 모델링 시트로 직접 사용하지 않음
- 얼굴, 의상 구조, 무기 구조는 캐릭터별 제작 시트에서 확정
- 승인된 개별 시트가 만들어지면 해당 시트를 최우선 기준으로 사용
