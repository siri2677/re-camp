# ART_DIRECTION.md

## 전체 아트 방향

Re:Camp의 아트 방향은 **귀여운 서브컬쳐 캐릭터 + 부드러운 포스트 아포칼립스**이다.

```text
밝고 귀여운 캐릭터
+ 자연에 잠식된 폐허 도시
+ 파스텔 톤 라이팅
+ 모바일 친화적 UI
```

## 캐릭터 스타일

- 미소녀/귀여운 캐릭터 중심
- 너무 무거운 밀리터리보다 가벼운 택티컬 스트리트 패션
- 후드, 고글, 백팩, 드론, 소형 무기 활용
- 실루엣은 명확하고 모바일 화면에서도 구분 가능해야 한다.

## 메인 캐릭터 초안

| 항목 | 내용 |
|---|---|
| 이름 | 루나 |
| 역할 | 정찰형 딜러 |
| 콘셉트 | 고양이 귀 후드 정찰병 |
| 무기 | 에너지 단검 |
| 성격 | 밝고 장난기 있지만 탐험 중에는 집중력이 높음 |
| 색상 | 민트, 크림, 핑크 포인트 |
| 기능 | 이동속도 증가, 자원 탐지, 회피 특화 |

## 배경 스타일

- 폐허 거리
- 오염된 숲
- 버려진 연구소
- 지하 상가
- 기계 무덤

배경은 어둡고 잔혹한 분위기보다 **아름답게 남겨진 폐허** 느낌을 우선한다.

## 컬러 톤

```text
메인: 민트 / 크림 / 라이트 퍼플
포인트: 핑크 / 네온 블루
위험 요소: 레드 / 딥 퍼플
배경: 그레이시 블루 / 파스텔 그린
```

## UI 스타일

- 둥근 버튼
- 가벼운 글래스/패널 느낌
- 모바일 터치 영역 확보
- 전투 중 UI는 최소화
- 보상 화면은 화려하고 명확하게

## 금지 스타일

```text
과도한 실사풍
지나치게 어둡고 잔인한 고어 표현
복잡한 UI 장식
모바일에서 식별 어려운 디테일
무거운 AAA 밀리터리 느낌
```

## 아트 우선순위

```text
1. 메인 캐릭터 루나 전신 콘셉트
2. 전투 화면용 SD/인게임 모델 방향
3. 폐허 거리 스테이지 배경
4. 몬스터 3종
5. UI 버튼/패널 스타일
6. 보상 아이콘
```
