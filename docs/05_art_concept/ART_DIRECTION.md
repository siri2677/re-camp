# Re:Camp Art Direction

> Version: 2.0
> 최종 갱신: 2026-07-24
> 상태: Active Design Baseline

## 1. 최종 비주얼 정의

Re:Camp의 비주얼은 **매력적인 성인 여성 서브컬처 캐릭터 + 자연에 잠식된 부드러운 포스트 아포칼립스 + 희망적인 캠프 생활**의 조합이다.
핵심 타깃은 캐릭터의 외형·성격·관계·성장에 애착을 느끼는 남성향 서브컬처 RPG 이용자다.

```text
2D 약 7등신 성인 서브컬처 캐릭터 일러스트
+ 3D 5~6등신 스타일라이즈드 인게임 캐릭터
+ 3D 스타일라이즈드 폐허와 캠프
+ 툰 셰이딩
+ 고정 Orthographic 쿼터뷰 전투
```

배경의 사실성보다 캐릭터의 매력, 식별성, 애착 형성과 모바일 플레이 가독성을 우선한다.

## 2. 표현 방식

| 구분 | 최종 방향 |
|---|---|
| 캐릭터 Key Art·대표 이미지 | 약 6.8~7.4등신, 기본 약 7등신의 성인 애니메이션 일러스트 |
| 캐릭터 제작 시트 | 2D 약 7등신 Front·Side·Back과 얼굴·의상·장비 상세 |
| 인게임 캐릭터 | 3D 5~6등신 스타일라이즈드 모델, 캐릭터별 약 5.2~5.8등신 |
| UI 초상화 | 2D 얼굴·상반신 중심, 헤어·눈매·대표색을 작은 크기에서 강조 |
| 카메라 | 고정 Orthographic 쿼터뷰 |
| 렌더링 | 툰 셰이딩 기반 3D |
| 배경 | 자연에 잠식된 스타일라이즈드 폐허 |
| UI | 둥근 패널, 가벼운 글래스, Android Landscape 가독성 |
| 분위기 | 위협적인 세계와 따뜻한 캐릭터·캠프의 대비 |

## 3. 캐릭터 디자인 핵심 원칙

### 3.1 한 문장으로 기억되는 디자인

각 캐릭터는 이름을 제외하고도 한 문장으로 설명 가능해야 한다.

- 루나: 고양이형 센서 후드와 쌍단검을 쓰는 폐허 정찰병
- 미유: 감정보다 드론이 더 솔직한 비대칭 테크니션
- 코코: 웃는 얼굴로 현장을 지휘하는 구조대장
- 이리스: 관측 장비처럼 정밀한 장총을 다루는 우아한 저격수
- 노아: 방패를 문처럼 펼쳐 이동식 피난처가 되는 수호자

### 3.2 패션과 기능의 결합

- 의상은 역할 코스튬이 아니라 도시형 패션 실루엣과 실제 기능 장비를 결합한다.
- 고양이 귀, 큰 소매, 케이프, 롱 코트, 대형 방패는 장식이 아니라 탐지·제어·구조·조준·방어 기능에 연결한다.
- 비대칭 절개, 레이어 차이, 한쪽 장비와 작은 포인트 컬러로 현대적인 인상을 만든다.
- 노출도는 캐릭터 매력·성격·활동성·3D 제작 가능성을 함께 검토하고 전원에게 동일하게 적용하지 않는다.
- 패션 요소가 전투 역할과 세계관을 가리면 제거한다.

### 3.3 성격의 반전 매력

항상 한 가지 표정과 성격만 보이는 캐릭터보다 평상시와 임무 상태의 대비를 우선한다.

- 루나: 장난스러움 ↔ 집중
- 미유: 무표정 ↔ 기계 이야기의 열정
- 코코: 따뜻한 미소 ↔ 현장 지휘
- 이리스: 완벽주의 ↔ 생활 실수
- 노아: 무뚝뚝함 ↔ 행동형 배려

### 3.4 공통 세계관 언어

- 전 캐릭터는 차콜·웜 그레이·크림 기반의 캠프 공용 소재를 일부 공유한다.
- 재활용 합성 섬유, 무광 보호대, 수리 스티치, 교체 가능한 장비 모듈을 사용한다.
- 장비는 마모됐지만 얼굴과 의상은 깨끗한 서브컬처 표현을 유지한다.
- 캠프 문장과 공용 체결 부품은 작게 반복하되 캐릭터별 대표 모티프를 침범하지 않는다.
- 기능 없는 발광선을 남발하지 않고 탐지·드론·치료·조준·방어 피드백에만 사용한다.

## 4. 2D 7등신과 3D 5~6등신의 분리

2D와 3D는 같은 캐릭터의 서로 다른 용도이며 단순 축소 관계가 아니다.

| 항목 | 2D 약 7등신 | 3D 5~6등신 |
|---|---|---|
| 목적 | 캐릭터 매력, 패션, 체형, 표정, 홍보 | 조작, 전투 가독성, 애니메이션, 성능 |
| 체형 | 긴 다리와 성인 여성의 패션 실루엣 | 머리·손·발·장비를 확대한 스타일라이즈드 체형 |
| 얼굴 | 미세한 눈매·입·표정과 화장 표현 | 눈매·헤어라인·대표 표정 중심 |
| 의상 | 레이어·절개·재질을 풍부하게 표현 | 큰 색면과 외곽 형태를 남기고 미세 장식 축소 |
| 헤어·천 | 흐름과 우아함을 충분히 표현 | 길이·물리 본·관통 위험을 줄임 |
| 장비 | 구조와 사용법을 상세하게 설명 | 쿼터뷰에서 종류·방향·상태가 즉시 읽히게 확대 |

- 2D 대표 일러스트의 기본 목표는 약 7등신이며 캐릭터별 약 6.8~7.4등신 범위를 허용한다.
- 3D는 캐릭터별 약 5.2~5.8등신을 사용하되 전체 허용 범위는 5~6등신이다.
- 2D 7등신은 허용되지만 얼굴이 실사 패션 모델처럼 지나치게 작아지면 수정한다.
- 3D에서 6등신을 초과해 길고 가늘어지거나 2~3등신 유아형 치비로 회귀하지 않는다.
- 2D 제작 시트마다 `2D to 3D Simplification` 항목을 작성한다.

## 6. 캐릭터별 실루엣 원칙

### 루나

- 접힘·회전이 가능한 고양이형 센서 후드
- 짧은 크림 재킷, 비대칭 허리 패널, 활동적인 하체
- 한 쌍의 에너지 단검과 꼬리처럼 보이는 스캐너 케이블
- 짧고 빠른 대각선과 전방 기울임 자세

### 미유

- 스모키 라일락 비대칭 보브와 한쪽 큰 작업용 소매
- 머리·어깨 주변을 도는 성격이 다른 드론 2기
- 큰 총기 대신 손·건틀릿·드론의 삼각 구도
- 구부정한 평상 자세와 장비를 지시하는 전투 자세

### 코코

- 코랄 웨이브 헤어와 짧은 구조대 하프 케이프
- 인젝터 배턴, 구조 하네스, 반투명 곡면 보호막
- 둥글고 안정적인 곡선 실루엣
- 평상시 포용 자세와 위기 시 명령 자세의 대비

### 이리스

- 길고 곧은 딥 플럼 헤어와 콜드 화이트 비대칭 롱 코트
- 키에 가까운 장거리 에너지 라이플
- 다섯 명 중 가장 긴 세로·대각선 실루엣
- 정렬된 후방 조준 자세

### 노아

- 미드나이트 네이비 짧은 보브와 허리선이 남는 방호 재킷
- 평상시 세로 케이스, 전투 시 문·벽이 되는 전개 방패
- 탄탄한 최장신 체형과 큰 수직 사각형 실루엣
- 정면을 막고 동료의 공간을 만드는 자세

## 7. 세계와 환경

### 배경 테마

- 폐허 거리
- 오염된 숲
- 버려진 연구소
- 지하 상가
- 기계 무덤
- 재건 중인 생존자 캠프

배경은 잔혹하고 절망적인 폐허보다 **시간과 자연에 의해 아름답게 남겨진 공간**을 지향한다.
캐릭터보다 디테일과 채도를 낮추고 이동, 위험, 자원, 상호작용 영역을 명확히 구분한다.

### 캠프

캠프는 전투 지역과 시각적으로 대비되는 안전한 장소다.

- 따뜻한 조명
- 천막, 안테나, 발전기, 작업대, 식량 저장고
- 캐릭터별 개인 공간과 생활 소품
- 수리 흔적이 보이지만 정돈된 구조
- 성장에 따라 증가하는 조명과 생활감

## 8. 컬러 언어

```text
프로젝트 공통 기반: 차콜 / 웜 그레이 / 크림
대표 희망색: 민트 / 코랄 / 앰버
기술 요소: 시안 / 네온 블루 / 퍼플
위험·정밀 요소: 크림슨 / 딥 플럼
배경: 그레이시 블루 / 파스텔 그린 / 웜 그레이
```

캐릭터와 공격 VFX는 배경보다 높은 명도·채도 대비를 가진다.
캐릭터별 포인트색은 큰 면적보다 얼굴·상체·대표 장비 주변에 집중한다.

## 9. 재질과 셰이딩

- 캐릭터: 깨끗한 툰 셰이딩, 얼굴 그림자 최소화
- 피부·얼굴: 부드러운 명암과 눈·입 중심의 표정 가독성
- 의상: 천, 합성 섬유, 무광 플라스틱, 제한적인 투명 소재
- 장비: 마모된 금속과 기능성 발광 패널
- 배경: 캐릭터보다 낮은 디테일과 채도
- 발광: 실제 기능과 전투 피드백에만 제한적으로 사용

## 10. UI와 VFX

### UI

- 둥근 버튼과 패널, 얇은 외곽선과 반투명 글래스
- Android Landscape에서 읽히는 글자와 충분한 터치 영역
- 전투 HUD 최소화
- 캐릭터별 포인트 컬러를 선택 화면과 스킬 UI에 반영
- UI 초상화는 얼굴·헤어·눈매·대표색만으로 5인을 구분

### VFX

- 루나: 청록의 짧고 날카로운 쌍단검 궤적과 센서 링
- 미유: 라일락·시안 홀로그램, 두 드론의 다른 탄도와 표정 신호
- 코코: 코랄·제이드 회복 파동과 반투명 곡면 보호막
- 이리스: 크림슨 조준선과 압축된 직선 타격광
- 노아: 앰버 격자 방벽과 무거운 반격 충격파

## 11. 트렌드 반영과 변형 규칙

- 최신 유행을 그대로 복제하지 않고 `패션 실루엣`, `기능 장비`, `반전 성격`, `표정 연출`로 분해해 적용한다.
- 캐릭터의 `LOCKED` 훅과 대표 장비를 유지하면 헤어 끝, 의상 절개, 노출도, 소품 배치는 `CONTROLLED` 범위에서 수정할 수 있다.
- 한 디자인 안에 유행 요소를 모두 넣지 않고 대표 특징 1개와 보조 특징 2개를 우선한다.
- SNS·스토어 썸네일에서는 얼굴과 한 문장 훅이 즉시 전달돼야 한다.
- 전신 후보는 흑백 실루엣, 128px 얼굴, 저채도, 장비 제거 상태로 교차 검토한다.
- 유행 반영으로 다섯 명이 동일한 검정 전술복·은발·네온 장식으로 수렴하면 실패다.

## 12. 금지 스타일

- 과도한 실사·무거운 AAA 밀리터리·잔혹한 고어
- 얼굴을 가리는 과도한 장비
- 모바일·쿼터뷰에서 읽히지 않는 미세 장식
- 캐릭터마다 다른 세계관처럼 보이는 디자인
- 다섯 명이 팔레트만 바꾼 동일 체형·얼굴·전술복으로 보이는 디자인
- 남성 플레이어블 캐릭터 또는 성별이 모호하게 읽히는 최종 디자인
- 2D에서 6.5등신 미만으로 지나치게 어려 보이는 대표 Key Art
- 3D에서 2~3등신 유아형 치비 또는 6등신 초과의 가늘고 긴 체형
- 외부 IP의 얼굴·의상·무기·로고를 직접 복제한 디자인
- AI 결과를 구조 검수 없이 최종 원화·3D 설계도로 사용

## 13. 레퍼런스 상태와 우선순위

- 생성 이미지는 WIP 또는 REVIEW이며 자동으로 APPROVED되지 않는다.
