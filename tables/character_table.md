# Character Table

| ID | 이름 | 역할 | 무기·장비 | 핵심 특징 | 해금 순서 | MVP 포함 |
|---|---|---|---|---|---:|---|
| CH001 | 루나 | 정찰형 근접 딜러 | 에너지 단검, 탐지 장치 | 빠른 이동, 3타 콤보, 자원 탐지 | 1 | Yes |
| CH002 | 미유 | 드론 원거리 딜러 | 전투 드론, 제어 건틀릿 | 자동 사격, 드론 강화, 자동 회수 | 2 | No |
| CH003 | 코코 | 회복·보호막 서포터 | 펄스 인젝터, 의료 장비 | 즉시 회복, 피해 흡수 보호막 | 3 | No |
| CH004 | 이리스 | 정밀 사격·보스 딜러 | 에너지 라이플, 조준 바이저 | 차지 사격, 표식, 보스 추가 피해 | 4 | No |
| CH005 | 노아 | 가드·반격 탱커 | 전개식 에너지 방패 | 정면 방어, 반격, 이동 차단 방벽 | 5 | No |

모든 캐릭터는 스토리와 탐험 진행으로 확정 해금한다.

| 캐릭터 | 해금 조건 |
|---|---|
| 루나 | 시작 캐릭터 |
| 미유 | 버려진 작업 구역 구조 임무 완료 |
| 코코 | 생존자 구호 임무 완료 |
| 이리스 | 장거리 신호 추적 임무 완료 |
| 노아 | 캠프 방어 스토리 완료 |
